# PRD 模板：标准产品需求文档

## 1. 产品概述
- **产品名称**：[简洁的产品名称]
- **一句话摘要**：[30字以内的价值主张]
- **问题陈述**：[描述要解决的核心问题]
- **目标用户**：[主要用户画像 + 次要用户画像]

## 2. 用户故事
以格式 "作为[用户类型]，我想要[功能]，以便[收益]" 列出优先级排序的用户故事。

### P0（MVP 必备）
- 作为[用户]，我想要[功能]，以便[收益]

### P1（重要）
- 作为[用户]，我想要[功能]，以便[收益]

### P2（锦上添花）
- 作为[用户]，我想要[功能]，以便[收益]
